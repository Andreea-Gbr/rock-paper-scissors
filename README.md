# ✊🖐️✌️ Rock, Paper, Scissors - AI Battle

✨ A Python-based implementation of the classic hand game, featuring an automated AI opponent.

## 🌟 Features
- 🛡️ **User-Friendly:** Case-insensitive input handling (e.g., typing "Rock" or "rock" both work).
- ♾️ **Infinite Loop:** Play as many rounds as you want without restarting the script.
- 🧹 **Input Filtering:** Automatically ignores invalid choices and asks you to try again without crashing.

## ⚙️ Prerequisites
- 🐍 Python 3.x

## 🚀 Installation & Usage

1. **⬇️ Download the script:**
   Save `rps_game.py` to your local machine.

2. **▶️ Run the script in your terminal:**
   ```bash
   python rps_game.py
   ```
